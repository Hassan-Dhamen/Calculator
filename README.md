# Calculator Web App

A simple,  calculator built with HTML, CSS, and JavaScript.
The calculator supports basic arithmetic operations and includes sound feedback on button clicks.

---

## Features

- Basic operations: Addition, Subtraction, Multiplication, Division
- Responsive design (desktop and mobile)
- Clean and modern user interface
- Button hover effects
- Reset (Clear) button
- Click sound on every button press
- Beginner-friendly JavaScript logic

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- HTML Audio API

---

## Project Structure

calculator-project/
├── index.html

---

## How to Run

1. Download or clone the project
2. Keep all files in one folder
3. Open `index.html` in any modern web browser
4. Start using the calculator

---

## Sound Feature

Each button click plays a short sound using the HTML Audio API.
The sound restarts on every click to allow fast interactions.

---

## Responsive Design

The calculator adapts to different screen sizes using Flexbox, CSS Grid, and media queries.
